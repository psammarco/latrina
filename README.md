# latrina-gentoo-overlay

This is a personal Gentoo overlay, and you are more than welcomed to use it as long as I am not sued for your grandpa incontinece. Sorry!

Overlay can be added to emerge through git and [repos.conf](https://wiki.gentoo.org/wiki//etc/portage/repos.conf)

```
[latrina]
location = /var/db/repos/latrina
sync-type = git
sync-uri = https://github.com/psammarco/latrina-gentoo-overlay.git 
priority=9999
```
